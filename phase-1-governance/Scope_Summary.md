# SOC 2 Scope Summary

## Overview
This document defines the scope of SOC 2 compliance for the AWS environment.

## Boundaries
- AWS Cloud resources under company ownership  
- Key AWS services: EC2, S3, IAM, CloudTrail, KMS, VPC  

## Exclusions
- On-premises resources  
- Third-party services not managed by AWS  

## Trust Services Criteria
- **Security**
- **Availability**
- **Confidentiality**
