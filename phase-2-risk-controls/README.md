# Phase 2 — Risk Assessment & Control Mapping

## Objectives
- Build SOC 2–aligned Risk Register.
- Map AWS controls to Trust Services Criteria (TSC).
- Start Control Register tied to risks.

## Tasks
- Excel → Risk_Register: add 8–12 risks (S3 exposure, IAM privilege creep, etc.).
- Excel → TSC_Control_Map: map AWS services to SOC 2 criteria (CC6.1, CC6.7, CC7.2/7.3, CC8.x).
- Excel → Control_Register: list controls, owner, frequency, evidence.

## Deliverables
- Updated Risk Register
- TSC Control Map
- Control Register
