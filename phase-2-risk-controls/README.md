# Phase 2 — Risk Assessment \& Control Mapping

## Objectives

* Build SOC 2–aligned Risk Register.
* Map AWS controls to Trust Services Criteria (TSC).
* Start Control Register tied to risks.

## Tasks

* Excel → Risk\_Register: add 8–12 risks (S3 exposure, IAM privilege creep, etc.).
* Excel → TSC\_Control\_Map: map AWS services to SOC 2 criteria (CC6.1, CC6.7, CC7.2/7.3, CC8.x).
* Excel → Control\_Register: list controls, owner, frequency, evidence.

## Deliverables

* SOC 2 Risk Register
* TSC Control Mapping Matrix
* Initial Control Register (with risk linkage)
